出征，带着英雄的嘱托

出征,带着英雄的嘱托

--"排雷英雄战士"杜富国为维和战友送行现场纪事

万里赴戎机，关山度若飞。12月5日凌晨两点，在浓浓的夜色中，第22批赴黎巴嫩维和多功能工兵分队离开营区豪迈出征，乘飞机赴黎巴嫩南部地区，执行为期一年的维和任务。

出征，亲人壮行。在为战友送行的队伍中，记者见到了“排雷英雄战士”杜富国的身影。得知战友跨国出征，杜富国坚持前来送行，用他的话说，“虽然身不能去，但心永远和战友们在一起”。

自受伤以来，杜富国始终惦记着扫雷事业。在与战友们的交流中得知，此次出征的维和多功能工兵分队，主要担负扫雷排爆、开辟通道等任务。离别之际，杜富国最关心的还是战友们的安全。“国际维和的是硝烟战场，直面的是生死考验。”在与战友刘学富聊天时，杜富国叮嘱他，“在危机四伏的任务区，你们要发扬大队传统，遵守操作规程，确保各项安全。”

刘学富和杜富国同年入伍，曾一同执行边境扫雷任务，是生死相依的好兄弟。杜富国受伤后，刘学富曾在病床边陪护数月。“富国，请放心，战友们进行了扎实充分的准备。”他告诉杜富国，由他们参与修订完善的30余项扫雷标准作业程序，已经被联合国推广使用；行前集训期间，他们通过了最严格的专业考核，取得了优秀成绩。

临行前，刘学富和战友们专门让杜富国试穿了新型防护服。“新型防护服更加轻便。”杜富国一边试一边说，“真为你们感到高兴。”刘学富说：“我们在一起时，我就是你的眼、你的手。出国执行维和任务后，我会把看到的一切和所做的一切，都原原本本告诉你，让你分享我们的感受。”

“越是走出国门，越要坚定信念。”作为参加此次任务的4名女兵之一，下士蒋承斩酝瓿扇挝癯渎信心。她告诉杜富国，自己是在高二时读过他的英雄事迹后萌发了参军入伍的念头，没想到还分配到了英雄所在的部队。“富国班长，当年您的一句‘你退后，让我来’，不知感动了多少莘莘学子，激励了多少热血青年。”得知杜富国正在练习毛笔字，蒋承蘸驼接衙亲门请杜班长写下“你退后，让我来”6个大字，“这次出国维和，我们将攻坚克难勇于担当，立起一流标准，擦亮中国名片。”

风雨兼程路，赤诚为和平。记者注意到，在这批多功能工兵分队中，有不少官兵已多次执行维和任务。扫雷二组组长、二级上士黄仕发告诉杜富国：维和是一扇“窗口”，我们一批批维和官兵牢记使命责任，以勇做刀尖舞者的血性胆气和无畏豪情，累计探排200余万平方米的疑似雷区，被联合国地雷行动中心誉为“世界上最好的扫雷队伍”……战友们的深情讲述，让杜富国听得热血沸腾。

轰鸣的发动机声中，维和官兵军姿挺拔、精神抖擞。送别之际，杜富国庄严敬礼，祝战友一路顺风，早日凯旋！

(本报昆明12月5日电)
